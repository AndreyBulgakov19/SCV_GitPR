# Краткая инструкция по работе с GIT
## Что есть GIT?
<image src="../git_pic.png" alt="картинка git">

*Git* - вариант работы в системе с контролем версий, в локальном и удаленном репозиториях

## Подготовка репозитория
Для создания локального репозитория нужно в необходимой папке в терминале выполнить команду **git init**,
и в этой папке появиться скрытая папка *.git* - создастся локальный репозиторий

## Создание комментариев
Для добавления изменений в файле нужно набрать команду
**git add <имя файла>**
Для комментария и сохранения изменений **git commit -m "комментарий"**

## Просмотр состояния репозитория
Для просмотра состояния репозитория используется команда **git status**,
по ее выполнению можно увидеть все изменения в файлах, если они были

## Создание ветки
Для создания версии файлов в отдельной ветке - нужно
использовать команду **git branch <имя ветки>**

## Переход между ветками
Для перехода между рабочими ветками есть команда **git checkout**
Здесь указывается ветка-рабочий файл, на которую нам необходимо переключиться
При выполнении этой команды мы перейдем в сохраненное состояние файлов, согласно версии комментария.

## Работа с репозиториями
1. Для работы с репозиториями необходимо зарегистрироваться на github.
2. Зайти на страничку с нужным репозиторием (методом поиска или по ссылке)
3. Нажать кнопку ***Fork***
4. Скопировать на локальный компьютер репозиторий комндой **git clone**
5. Cоздать новую ветку **git branch <new>** 
6. Перейти в эту ветку. Внести необходимые изменения
7. Сохранить командой **git commit -m "комент  "**
8. Сохранить в своем репозитории
9. Отправить запрос на слияние веток (добавления своей версии) командой **pull request**
