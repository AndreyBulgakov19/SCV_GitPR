# Инструкция по работе с удалёнными репозиториями

## 1. Копирование удалённого репозитория себе на компьютер
* Чтобы клонировать удалённый репозиторий, необходимо на GitHub открыть страницу проекта, который мы хотим скопировать, после нажать на зелёную кнопку `Code`, в открывшемся окне выбрать `Local`, `Https`, после нажать на значок `копировать`. Далее нужно открыть папку, в которую будет копирован удалённый репозиторий и прописать комманду:
```
git clone <ссылка, которую мы скопировали во вкладке "Code">
```
* После этого вы можете работать с данным репозиторием на своём компьютере.
