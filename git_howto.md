# Подсказки по Git

Создание репозитория:
```sh
git init
```
Добавить к индексации:
```sh
git add
```
Зафиксировать изменения:
```sh
git commit -m "Message"
```
Посмотреть лог в полном виде:
```sh
git log
```
Посмотреть лог в сокращенном виде:
```sh
git log --oneline
```

Отображение всех веток
```sh
git branch
```
Перемещение по веткам
```sh
git checkout
```

Как переименовать ветку master:
```sh
git branch -M <branch name>
```

Как поставить нужный репозиторий в качестве удаленного репозитория по умолчанию:
```sh
git remote add origin <repository link>
```

Как вписать новые изменения с локального в удаленный репозиторий:
```sh
git push -u origin main
```

Как вписать новые изменения с удаленного в локальный репозиторий:
```sh
git pull
```
